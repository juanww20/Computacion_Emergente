# PROYECTO 2: Optimización de Horarios Académicos

## Descripción
Diseñar un sistema de **optimización de horarios** usando Algoritmos Genéticos,
evitando solapamientos de materias, aulas y profesores.

## Requisitos mínimos
- Representación del individuo (horario)
- Función fitness basada en conflictos
- Restricciones duras y blandas
- Uso opcional de lógica difusa para preferencias

## Restricciones duras
- Un aula no puede tener dos clases al mismo tiempo
- Un profesor no puede dictar dos clases simultáneamente

## Restricciones blandas (opcionales)
- Preferencias horarias
- Distribución equilibrada de materias

## Entregables
- Documento explicativo
- Ejemplo visual de un horario con conflictos y su mejora
